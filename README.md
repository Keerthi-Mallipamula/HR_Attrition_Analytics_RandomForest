# HR_Attrition_Analytics_RandomForest
This project analyzes employee attrition data to identify &amp; predict which employees are likely to leave. Using Python and machine learning, it includes data cleaning, exploratory analysis, and model building to derive insights that can help organizations improve retention strategies.
Tech Stack

Language: Python

Libraries: Pandas, NumPy, Matplotlib, Seaborn, Scikit-learn

Environment: Jupyter Notebook

Dataset: IBM HR Employee Attrition

Workflow

Data cleaning and preprocessing

Exploratory data analysis (EDA)

Feature engineering

Model training and evaluation


Key Insights

Attrition strongly relates to factors like OverTime, JobSatisfaction, and MonthlyIncome.

Classification models accurately predict employee attrition.
